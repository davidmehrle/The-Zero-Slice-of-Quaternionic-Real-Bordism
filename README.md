This repository contains the code and charts accompanying the paper *The Zero Slice of Quaternionic Real Bordism* by Bert Guillou, Jesse Keyes, and David Mehrle. The code is a SageMath worksheet (iPython notebook) called `Q8norms.ipynb`. 
- If you simply want to view the contents of the file, there is a convenient online ipython viewer here: https://ipynb.js.org.
- If you want to run the code yourself, you can do so:
  - in the browser using CoCalc (https://cocalc.com), or
  - on your machine by installing sage (https://doc.sagemath.org/html/en/installation/index.html).
